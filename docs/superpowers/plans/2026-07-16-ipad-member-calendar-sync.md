# iPad 成员与日历联动 Implementation Plan

**Goal:** iPad 使用管理在切换成员和选择日历日期时始终展示正确日期、正确成员的使用记录。

**Architecture:** 在全局 `state` 增加 `ipadSelectedDate`，它只负责 iPad 视图的查询上下文。`renderIpad` 使用该日期筛选额度和使用记录；成员切换统一将它重置为当天。日历单元用按钮承载日期，点击后更新该状态并重新渲染。

## 实施步骤

1. 以失败测试固定 iPad 独立日期状态与切换成员时的重置行为。
2. 以失败测试固定可点击日历单元、记录区日期标题和按日期查询的行为。
3. 将历史日期置为只读，保留今日的创建逻辑。
4. 运行完整 Node 测试与差异检查。
